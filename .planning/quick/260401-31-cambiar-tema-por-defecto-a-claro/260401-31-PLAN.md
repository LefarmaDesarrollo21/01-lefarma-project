# Quick Task 260401-31: Cambiar tema por defecto a claro cuando no hay preferencias guardadas

**Created:** 2026-04-01  
**Status:** Ready for execution

## Task Description

Por defecto al iniciar el sistema y no tiene nada guardado aún, por defecto inicia el sistema con tema claro.

## Context

Actualmente el tema por defecto en `configStore.ts` es `'system'`, lo que significa que el sistema sigue la preferencia del sistema operativo del usuario. Se requiere cambiar el valor por defecto a `'light'` para que siempre inicie con tema claro cuando no hay preferencias guardadas previamente.

## Files to Modify

| File | Action | Description |
|------|--------|-------------|
| `lefarma.frontend/src/store/configStore.ts` | Update | Cambiar `tema: 'system'` a `tema: 'light'` en `DEFAULT_UI_CONFIG` (línea 11) |
| `lefarma.frontend/src/store/configStore.ts` | Update | Cambiar `tema: 'system'` a `tema: 'light'` en función `resetConfig()` (línea 248) |
